# Research Companion（モニター版）

研究（/課題/試験勉強）を「今日やること→実行→振り返り」で回し続けるための、軽量な伴走アプリです。  
入力はソリッドに、フィードバックはやさしく、継続しやすい体験を目指しています。

研究をする際に必要な自分の管理，難しいですよね。何をすればいいのかも分からなくなることもあるし，何から手をつければいいかも分からなくなることもある。かといって，何かしらのプログラムを組んで管理をするとかも面倒だし，そこまでの苦労をするなら自分で雑に管理する。

そんな方に向けた，何をどれくらいの時間取り組むか，そして，それが達成できたかを一括で管理できるアプリを作ってみました。

シンプルに「今日何をするかを決める→実行する→振り返って明日に引き継ぐ」のサイクルを回し続けるためだけのアプリです。余計な機能はいらない。ただ宣言して，実行して，記録する。それだけのシンプルな構成にしてあります。

---

## できること（現状）
- 「今日することをまとめる」：今日のタスクを短く追加、目標時間を設定できます。また，作業内容のカテゴリに応じて，低ステップの目標設定を自動で提案します。
- 「今日」：タスク開始でタイマーを動かせます（中断・再開・完了）。
- 「今日を終える」：1日のまとめを保存できるようにしました（サイドバーに記録されます）。
- 遠い目標：遠い期日の目標に向けた「今日すること」を決めるための設定をすることができます。
- 週間レポート（試作）：2週間ごとを最長として，毎日の記録をレポートにしてダウンロードできるようにしました。
